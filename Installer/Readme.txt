BIENVENIDO A GESTIONTIME DESKTOP

Este instalador configurará GestionTime Desktop 1.1.0 en su sistema.

GestionTime Desktop es una aplicación moderna para la gestión de tiempo y 
partes de trabajo con las siguientes características:

✓ Interfaz moderna con WinUI 3
✓ Gestión completa de partes de trabajo
✓ Conexión con APIs REST
✓ Reportes y gráficas detalladas
✓ Compatible con Windows 10/11

REQUISITOS DEL SISTEMA:
- Windows 10 versión 1809 o superior (x64)
- 512 MB de memoria RAM
- 300 MB de espacio libre en disco
- Conexión a Internet (para funcionalidad completa)

CARACTERÍSTICAS DE ESTA INSTALACIÓN:
- Instalación self-contained (sin dependencias externas)
- Runtime .NET 8 incluido
- WindowsAppSDK incluido
- Configuración automática

El proceso de instalación copiará todos los archivos necesarios y 
configurará accesos directos en el menú inicio.
